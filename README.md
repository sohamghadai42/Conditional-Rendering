# React Conditional Rendering Learning Project

A hands-on learning exploring conditional rendering techniques in React with Tailwind CSS styling.

##  What I'm Learning

- Conditional rendering in React
- Component creation and structure
- Tailwind CSS for styling
- React fundamentals

##  Technologies Used

- React
- Tailwind CSS
- JavaScript (ES6+)


##  Getting Started

1. Clone the repository
```bash
git clone https://github.com/sohamghadai42/Conditional-Rendering.git
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm start
```
